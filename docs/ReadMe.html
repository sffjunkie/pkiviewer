PKI Viewer

Displays information in the terminal from the following PKI file types

  • Certificates
  • PKCS#12 files
  • Certificate Signing Request (Coming Soon™)
  • Certificate Revocation List (Coming Soon™)

Also Coming Soon™ is Certificate Transparency information.

Features

  1. Files can be in either PEM or DER formats

  2. If a URL is passed a certificate can be downloaded automatically

  3. The certificate information can be output to the terminal, an SVG file or an HTML file.

  4. The information that is displayed can be configured (either hidden, displayed dim or highlighted)

  5. The colors can be configured

  6. Uses rich for the fancy output/SVG & HTML generation and cryptography for the low level parsing.

DER format certificates should have the extension '.cer' and PKCS#12 '.p12'

PKCS#12 Files

If the PKCS#12 file is encrypted you will be prompted to enter the passsword

Usage

Output certificate to the terminal

pkiviewer testcert.pem

or

pkiviewer https://bbc.com

pkiviewer https://bbc.com

Output the certificate to an SVG file

pkiviewer --save-svg=cert.svg testcert.pem

Output the certificate to an HTML file

pkiviewer --save-html=cert.html testcert.pem

For SVG and HTML output the number of columns to output can be set with the --width option

Display Configuration

pkiviewer looks for a pkiviewer.toml in the current directory.

This file can configure what information output and the colors.

Colors are configured in a colors section and the information visibility in a visibility section.

Colors

The default colors are

default = white
error = red
warning = yellow
info = green
key = blue
value = white
extension_critical = yellow
